# RubiCube
Game - Simulator of Rubiks Cube

Packages :

    PyGame - 'py -m pip install -U pygame --user'

Running : start 'main.py' file

Keys :

    1)camera view:
    
        mode    = 'o'
        rotate  = 'MMB + mouse'
        fov     = + 'MMB4'; - 'MMB5'
        zoom    = + '+'; - '-'
        
    2)cube moving languge:
    
        (+ U) - 'w' || (- U) - 'shift + w'
        (+ D) - 's' || (- D) - 'shift + s'
        (+ F) - 'e' || (- F) - 'shift + e'
        (+ B) - 'q' || (- B) - 'shift + q'
        (+ R) - 'd' || (- R) - 'shift + d'
        (+ L) - 'a' || (- L) - 'shift + a'
